# Style Guide Bahasa Indonesia - MutuGemba

Dokumen ini menjadi acuan istilah UI agar konsisten lintas layar laporan dan dashboard.

## Istilah Utama

- `Ringkas`: untuk tampilan singkat/preview (bukan `Preview`).
- `Lengkap`: untuk tampilan penuh/detail (bukan `Full`).
- `Subtotal`: tanpa tanda hubung (bukan `Sub-total`).
- `Total NG`: tetap dipakai sebagai istilah utama untuk kuantitas cacat.
- `Rasio NG`: dipakai untuk persentase NG.
- `Belum Diinput`: status data belum terisi (bukan `Belum Input`).
- `Akhir Pekan/Libur`: status hari non-produksi.
- `Nomor Dokumen`: label dokumen formal (bukan `No. Dokumen`).

## Aturan Penulisan

- Gunakan Bahasa Indonesia untuk label UI operator.
- Pertahankan istilah domain yang sudah baku di lantai produksi (`NG`, `OK`, `PIC`, `Line`).
- Hindari campuran sinonim untuk konteks yang sama di layar berbeda.
- Gunakan judul singkat dan jelas untuk komponen tabel dan kartu metrik.
